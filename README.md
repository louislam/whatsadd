# WhatsAdd

This is an Android small useful app which I developed in 2017. Now I decide to open source it!

- Chat with anyone on WhatsApp without adding them into your contact list
- Fancy UI (Still look great in 2022!)
- 500K+ Downloads
- High Rated 4.5 ⭐
